## FAQ

### What is this?

This is a public, crowd-sourced bibliography about the history of natural language processing (NLP). It started as a [Bluesky thread]() and grew from there.

### Can I add to this?

Yes please! If you know of books, websites, interviews, blogs, social media threads, or other documentary media that we should archive here, please add it!
